# 3주차 모각코 공부 내용

### C++ programming

***pointer type 변수*** 

ex> 
```
int i = 97;
int *p = &i; // i에 대한 주소 참조
```
<b>역참조 연산자 *</b>

실행문에 포인터 타입 변수 앞에 *을 써주면 그 포인터 타입 변수가 리턴하는 주소의 역참조값을 반환함.
```
int i = 97;
int *p = &i;

cout << p << endl; // p = i의 주소값
cout << *p << endl; // *p = 97
```

### 데이터베이스

관계 대수

1. 집합 연산
2. 순수 관계 연산 (셀렉트, 프로젝트)

셀렉트 -> 조건식에 맞는 튜플들의 집합
프로젝트 -> 속성값에 맞는 데이터 리턴
